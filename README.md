# 🎉 Happy Birthday Surprise App 🎉

A beautiful, interactive birthday website built with React and Vite, featuring:

## ✨ Features
- **Romantic Dark Theme** - Elegant dark gradient background
- **Falling Hearts Animation** - Continuous heart rain effect
- **Interactive Photo Gallery** - Photos appear with cute animations
- **Typewriter Birthday Message** - Letters appear one by one
- **Surprise Page** - Confetti and special birthday wishes
- **Responsive Design** - Works perfectly on mobile and desktop

## 🚀 Quick Start

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🎨 Technologies Used
- React 19
- Vite
- Framer Motion (animations)
- React Confetti
- React Icons
